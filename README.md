# omame's React boilerplate
## includes
- MIT license
- TypeScript
- Tailwind CSS with twin.macro
- ESLint
- Webpack 5
- Babel

## how to use
### linting
- `yarn lint`
- `yarn lint:fix`

### building
- `yarn build`
- `yarn build:production`

### testing with Webpack Dev Server
- `yarn start` (uses port 9000)
